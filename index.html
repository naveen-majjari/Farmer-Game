Farmer Harvest Game

Collect crops before the timer runs out. Avoid obstacles. Use power-ups when you see them!

  • Move with Arrow Keys or WASD
  • P to pause

About this build

This version demonstrates ES6 modules, arrow functions vs bind for this binding, levels, configurable difficulty via JSON, and multiple crop types with different points.

Note: Because it uses fetch() to load assets/config.json, run from a local server (e.g., python -m http.server) rather than opening the HTML file directly.
